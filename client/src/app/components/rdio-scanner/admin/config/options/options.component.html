Optional prompt to bias Whisper toward domain vocabulary. Used for systems that don't have their own prompt set. Hard cap 896 characters — longer values are truncated server-side (trailing portion kept).

Max Transcriptions / Minute (per key)
Per-key cap on requests in the last 60 seconds. Groq enforces 30/min per key, so set this to 30 (or a couple below to leave headroom). Effective throughput is this × number of keys. 0 disables the cap. When every key is at its limit or paused on a 429 backoff, incoming calls are skipped (never queued).

Min Audio Bytes
Skip transcription for clips smaller than this many bytes. Tiny clips are usually squelch bursts or mic-keys that waste quota. 0 disables the filter; 8000 ≈ 2 s of audio at typical rdio bitrates.

Wait For Transcript
When enabled, every client holds each call's audio playback for up to 15 seconds until its transcript is available, then plays. If Whisper doesn't return in time the call plays without a transcript (no calls are lost). Useful if you want viewers to read along while listening. Applies to all clients globally.
